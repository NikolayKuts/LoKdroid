LogManager

internal class LogManager(val minLevel: Level, val logger: Logger, val formatter: Formatter, val tagProvider: () -> String, val logBuilderProvider: LogBuilderProvider)

Manages logging operations by filtering based on log level, formatting messages, and delegating the logging to a specific logger implementation.
