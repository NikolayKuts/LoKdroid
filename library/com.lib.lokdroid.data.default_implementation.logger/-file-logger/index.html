FileLogger

class FileLogger(filePath: String, fileName: String = DEFAULT_FILE_NAME) : Logger

A file-based logger that writes log messages to a specified file in an asynchronous manner. This logger appends each log entry to the end of the file, formatting each entry with a timestamp, log level, and tag.

Parameters

filePath

The directory path where the log file will be created or exists.

fileName

The name of the file to write logs to. Defaults to DEFAULT_FILE_NAME.
